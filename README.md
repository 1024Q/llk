# 连连看 HTML 版

一个无需安装依赖的单文件连连看小游戏。直接用浏览器打开 `index.html` 即可开始。

## 功能

- 三种难度：简单、普通、困难
- 支持提示、洗牌、重新开始
- 计时、得分、剩余牌数、连击统计
- 关卡模式：通关后自动进入下一关，分数累计
- 本地最高分：使用浏览器存储记录历史最高分
- 暂停/继续：暂停时冻结倒计时和棋盘操作
- 音效开关：匹配、失败、提示、洗牌、通关都有反馈音
- 转弯设置：默认最多转弯 2 次，可调整到最多 5 次
- 关卡结算：每关完成后展示用时、奖励、总分和最长连击
- 道具系统：炸弹、冻结、放大镜可辅助通关
- 每日挑战：每天生成固定牌局，单独记录每日最高分
- 分项排行榜：按难度、每日挑战、最高关卡、最快通关和最长连击记录

## 文件结构

- `index.html`：页面结构
- `css/style.css`：界面样式和动画
- `js/game.js`：游戏逻辑

## 本地运行

双击 `index.html`，或在浏览器中打开该文件。
